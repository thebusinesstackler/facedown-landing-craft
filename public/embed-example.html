Recovery Equipment Order Form - Embed Example

How to Embed This Form

To embed this order form on your website, follow these simple steps:

Step 1: Add the container div

<div id="recovery-order-form"></div>

Step 2: Include the embed script

<script src="https://yourdomain.com/embed-script.js"></script>

Complete Example:

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Order Your Recovery Equipment</h1> <!-- Order form will appear here --> <div id="recovery-order-form"></div> <!-- Include the embed script --> <script src="https://yourdomain.com/embed-script.js"></script> </body> </html>

Custom Options (Optional):

You can customize the form by calling the embed function with options:

<script> embedRecoveryOrderForm({ containerId: 'my-custom-container', width: '100%', height: '900px' }); </script>
